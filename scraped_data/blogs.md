## Can Your Old Phone Run AI? The Ultimate Compatibility List
*Source: https://sagartamang.com/blog/best-android-llms*

A compatibility tier-list for running LLMs locally. 
- **Tier 1 (3-4GB RAM):** Qwen 2.5 0.5B, Llama 3.2 1B. 
- **Tier 2 (6GB RAM):** DeepSeek Coder 1.3B, Phi-3.5 Mini. 
- **Tier 3 (8GB+ RAM):** Llama 3.2 3B, Mistral 7B.
